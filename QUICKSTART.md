# 🚀 クイックスタートガイド

## すぐに始める（3ステップ）

### 1️⃣ BASEからCSVをダウンロード
```
BASE管理画面 → 注文管理 → CSVダウンロード
```

### 2️⃣ このWebサイトで変換
```
ファイルをドラッグ&ドロップ → 変換ボタンをクリック → ダウンロード
```

### 3️⃣ クリックポストにアップロード
```
https://clickpost.jp/ → まとめ申込 → CSVアップロード
```

---

## 📦 デプロイ手順（初心者向け）

### Render.com（完全無料・簡単）

1. **Renderアカウントを作成**
   - https://render.com/ にアクセス
   - 「Get Started」をクリック
   - GitHubアカウントで登録

2. **GitHubにコードをアップロード**
   - GitHubで新規リポジトリを作成
   - このプロジェクトをアップロード

3. **Renderでデプロイ**
   - Renderダッシュボードで「New +」→「Web Service」
   - GitHubリポジトリを選択
   - 設定：
     * **Name**: base-to-clickpost（任意の名前）
     * **Environment**: Python 3
     * **Build Command**: `pip install -r requirements.txt`
     * **Start Command**: `python app.py`
   - 「Create Web Service」をクリック

4. **完了！**
   - 数分後に自動的にデプロイされます
   - 提供されたURLをブックマーク

## ⚠️ 重要な注意事項

### ✅ できること
- BASEの注文CSVをクリックポスト形式に変換
- 最大40件までの注文を一度に処理
- ドラッグ&ドロップで簡単アップロード
- スマホ・タブレットからも利用可能

### ❌ できないこと
- 注文データの保存（セキュリティのため）
- 40件以上の一括変換（クリックポストの制限）
- BASE以外のプラットフォームのCSV変換

### 🔒 セキュリティ
- アップロードされたファイルは処理後すぐに削除されます
- サーバーにデータは一切保存されません
- 完全に安全です

## 🆘 トラブルシューティング

### エラー: 「変換中にエラーが発生しました」
**原因**: ファイル形式が正しくない可能性があります
**解決策**:
1. BASEから再度CSVをダウンロード
2. ファイルをExcelで開かずに直接アップロード
3. ファイルサイズが5MB以下か確認

### エラー: 「ファイルが選択されていません」
**原因**: ファイルが正しくアップロードされていません
**解決策**:
1. ブラウザをリロード
2. もう一度ファイルを選択

### 文字化けする
**原因**: Excelで直接開いている可能性があります
**解決策**:
1. Excelで「データ」→「テキストファイル」を選択
2. エンコーディングで「Shift-JIS」を選択
3. または、クリックポストに直接アップロード（推奨）
